# 从这里开始

你现在基础接近零基础，目标是 2026-06-30 周二上机考试能写出完整程序。不要先啃所有课件，先按下面顺序走。

## 今天先做这 4 件事

1. 打开 `visual_studio_workflow.md`，照着新建一个 C++ 控制台项目。
2. 打开 `src/exam_template.cpp`，把内容复制到你的 `main.cpp`，运行一次。
3. 不看答案，做 `exercises.md` 的练习 1：温度统计。
4. 卡住超过 15 分钟，再对照 `src/01_foundation_temperature.cpp`。

- 会新建 Visual Studio C++ 控制台项目。
- 会运行和调试。
- 会写 `cin`、`cout`、`if`、`for`、函数、`vector<int>`。
- 能独立写出一个数组统计程序。

## 明天必须达到的状态
